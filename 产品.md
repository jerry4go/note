#### 硅基流动

硅基流动是**国内头部第三方 MaaS（模型即服务）+ AI 推理基础设施厂商**，定位「Token 工厂」，不自研基础大模型，专门聚合全网开源 LLM、多模态 AIGC 模型，靠自研加速引擎做低成本、高并发推理服务。

两大核心产品：

1. **SiliconCloud（公有云平台）**：线上 API 调用市场，160 + 开源模型一键调用，兼容 OpenAI 标准接口；
2. **SiliconLLM 推理引擎**：底层加速内核，可私有化部署，大幅提升模型推理速度、降低显存占用；配套 OneDiff 图像加速库支撑文生图 / 视频 AIGC。
   
   

官网地址  https://cloud.siliconflow.cn/me/models



#### Anything LLM

AnythingLLM 是 Mintplex Labs 开发的**开源一体化本地优先 RAG 知识库 + 轻量 Agent 桌面 / 容器应用**，主打**全本地隐私、文档对话、轻量化私有化 AI**。

定位：**个人 / 小团队私有文档 AI 助手**，开箱即用，自带桌面客户端，不用复杂部署，核心口号：Any LLM、Any Document、Any Agent、Fully Private。

核心特点：默认所有文档、向量、对话数据本机存储，不上传第三方，极致隐私。
